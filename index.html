TinkerCAD Projects

This page shows some of the circuit projects that I was involved in during my college which involved the use of the infamous TinkerCAD platform.

Description :
This Tinkercad project can be implemented in Blind Sticks for visually challenged people. This blind stick has many uses.
  • This circuit when placed correctly, can detect an object or a wall in front of the stick and alert the user.
  • This circuit has the potential to detect if their is a downward staircase or a hole and will alert the user.
  • This circuit can also be implemented to detect water bodies near the path so that if the user can avoid them.

Components Used:
  • Arduino UNO
  • Ultrasonic Distance Sensor
  • Passive IR Sensor
  • Buzzer
  • Vibration Motor
  • NPN Transistor
  • Moisture Sensor (here, denoted by the use of a push button)
  • Resistor
